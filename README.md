ISLE TOOLS is an all-in-one desktop utility built specifically for The Isle (Evrima).

instead of going into the game just to change your graphics and coming back out, 
you apply a preset in seconds and youre done. LOW, MID, HIGH one click presets that 
actually make sense. things like shadows always maxed, motion blur set right, clouds 
at max since they cost nothing, materials at 0 because they should always be 0.

not just a graphics tool though. you can browse live servers pulled from Battlemetrics 
and Steam, see player counts and ping, star your favourites, and connect directly 
from the app. if you know a server's IP like Petits Pieds or any private server, 
you can add it manually and it saves permanently.

there's also a notes page for your base coords, server IPs, dino strats, whatever 
you wanna keep track of. logs into your Steam account automatically if Steam is running. 
backs up your config every time you change something so you can always roll back.

built in Python, runs as a standalone .exe, no install needed.
